Symfony 4 demo project
======================

This is demo project for learning the Symfony 4 framework.

Installation
------------

1. Clone project

    `$ git clone https://github.com/greeflas/Symfony4.git symfony4-demo`
    
2. Go to project dir
    
    `$ cd symfony4-demo`
    
3. Run server

    `$ docker-compose build && docker-compose up`
